Class: G3XInternalPrimaryFlightPlanRouteProvider

A provider of flight plan routes representing the structure of the internal primary flight plan of an instance of G3XFms.

getRoute()

getRoute(): Promise<object>

Gets a flight plan route representing the structure of the internal primary flight plan loaded in this provider's G3XFms instance.

Returns

Promise<object>

A Promise which is fulfilled with the flight plan route representing the structure of the internal primary flight plan loaded in this provider's G3XFms instance. The returned Promise is guaranteed to never be rejected.
