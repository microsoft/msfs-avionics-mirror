Class: ValueTouchButton<S>

A touchscreen button which displays the value of a bound state.

The root element of the button contains the touch-button-value CSS class by default, in addition to all root-element classes used by TouchButton.

The value of the button's bound state is rendered into a child div element containing the CSS class touch-button-value-value.
