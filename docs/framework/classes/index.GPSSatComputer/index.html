Methods​

acquireAndUseSatellites​

▸ acquireAndUseSatellites(): void

Instantly acquires and starts using all satellites with sufficient signal strength. If signal strength allows, SBAS satellites are instantly promoted to the Acquired state, and GPS satellites are instantly promoted to the InUse/InUseDiffApplied state.

If this system is not initialized, the operation will be delayed until just after initialization, unless reset() is called between now and then.

Has no effect if this system is a replica.

Returns​

void

reset​

▸ reset(): void

Resets the GPSSatComputer system. This will set the of the system to Searching and the state of every satellite to None.

If this system is not initialized, this method has no effect other than to cancel any pending operations triggered by previous calls to acquireAndUseSatellites().
