Enumeration: StartupScreenPrebuiltRow

Keys defining pre-built MFD startup screen rows.

Enumeration Members​

Navigation​

• Navigation = "Navigation"

The navigation row. Displays the expiration date of the current navigation database. The row will be rendered in amber if the simulation time is past the expiration date.

Defined in​

workingtitle-instruments-g3000/html_ui/Pages/VCockpit/Instruments/NavSystems/WTG3000/MFD/Components/Startup/StartupScreenRow.ts:41

System​

• System = "System"

The system row. Displays the name of the airplane as the title and the avionics system version as the value.
