NameTypeDefault valueDescription
contextMapSystemContext<MapFlightPlanFocusRTRControllerModules, any, any, MapFlightPlanFocusRTRControllerContext>undefinedThis controller's map context.
nominalFocusMarginsSubscribable<Readonly<Omit<Float64Array, "set" | "copyWithin" | "sort">>>undefinedA subscribable which provides the nominal focus margins, as [left, top, right, bottom] in pixels. The nominal margins define the offset of the boundaries of the focus region relative to the map's projected window, excluding the dead zone. Positive values represent offsets toward the center of the window. When the flight plan is focused, the focused elements of the plan are guaranteed to be contained within the focus region.
defaultFocusRangeIndexnumberundefinedThe default map range index to apply when the flight plan focus consists of only a single point in space.
