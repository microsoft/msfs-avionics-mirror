  • GarminAPConfig

Constructors​

new G3000APConfig()​

new G3000APConfig(bus, fms, lnavComputer, verticalPathCalculator, navToNavComputer, config, pluginOptions): G3000APConfig

Creates a new instance of G3000APConfig.

Parameters​

ParameterTypeDescription
busEventBusThe event bus.
fmsFms<any>The FMS instance from which to source data.
lnavComputerLNavComputerThe LNAV computer from which to source data.
verticalPathCalculatorVNavPathCalculatorThe vertical path calculator to use for the autopilot's internal VNAV and glidepath computers.
navToNavComputerGarminNavToNavComputerThe nav-to-nav computer from which to source data.
configAvionicsConfigThe avionics configuration object.
pluginOptionsreadonly G3000AutopilotPluginOptions[]An array of autopilot options defined by plugins. The array should be ordered such that options from plugins that were loaded earlier are positioned before options from plugins that were loaded later.

Returns​

G3000APConfig
