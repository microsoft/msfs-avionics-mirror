Methods​

buildSubTree​

▸ Private buildSubTree(nodeIndex, pivotDimension, start, end): void

Builds a portion of this tree starting from a specified node using the element indexes stored in a specified section of this tree's index arrays. The built subtree is guaranteed to be balanced. Before calling this method, the index array at position 0 should contain keys sorted in the specified pivot dimension, the array at position 1 should contain keys sorted in the dimension after the pivot dimension, etc (with the dimension wrapping back to 0 when reaching this.dimensionCount).

Parameters​

NameTypeDescription
nodeIndexnumberThe index of the tree node at which to start building the tree. The element associated with the pivot key will be placed at this node.
pivotDimensionnumberThe dimension in which to split the first level of the tree built by this method.
startnumberThe first index, inclusive, of the section of this tree's index arrays to use to build the tree.
endnumberThe last index, exclusive, of the section of this tree's index arrays to use to build the tree.

Returns​

void
