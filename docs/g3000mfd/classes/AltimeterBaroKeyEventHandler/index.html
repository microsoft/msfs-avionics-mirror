changeBaroSetting​

▸ Private changeBaroSetting(info, direction): void

Increments or decrements the baro setting for an altimeter. If STD BARO mode is currently on and baro preselect is supported, the preselected baro setting will be changed instead of the active baro setting. If STD BARO mode is currently on and baro preselect is not supported, STD BARO will be deactivated and the active baro setting will be changed.

toggleBaroStd​

▸ Private toggleBaroStd(info): void

Toggles STD BARO mode for an altimeter. If STD BARO is currently off, this will turn on STD BARO mode, set the preselected baro setting to the current active baro setting, and set the active baro setting to 29.92 inches (1013 hPa). If STD BARO is currently on, this will turn off STD BARO mode and set the active baro setting to the current preselected baro setting.
