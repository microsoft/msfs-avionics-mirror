nominalFocusMarginsSubscribable<Readonly<Omit<Float64Array, "set" | "copyWithin" | "sort">>>undefinedA subscribable which provides the nominal focus margins, as [left, top, right, bottom] in pixels. nominalFocusMargins​

• Private Readonly nominalFocusMargins: Subscribable<Readonly<Omit<Float64Array, "set" | "copyWithin" | "sort">>>

A subscribable which provides the nominal focus margins, as [left, top, right, bottom] in pixels. The nominal margins define the offset of the boundaries of the focus region relative to the map's projected window, excluding the dead zone. Positive values represent offsets toward the center of the window. When the flight plan is focused, the focused elements of the plan are guaranteed to be contained within the focus region.
