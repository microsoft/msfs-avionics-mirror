Class: NavBaseFields

These are the fields that should be exposed by NavSources and NavIndicators. Not all fields will be used by all the different Nav Sources. To add a new field, just add it to NavBaseFields, then add some code to each of the different NavSources to keep that field updated. Don't use this class directly. Every Subject should be able to be set to null, so that when the source is set to null, all the fields cna be nulled. This isn't abstract so that we can instantiate it in one place to help generate setters.
