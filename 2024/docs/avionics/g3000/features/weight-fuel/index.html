G3000 Weight and Fuel

Introduction​

The G3000 offers a weight and fuel feature which supports basic weight and fuel calculations. Using the GTC Weight and Fuel page (see image below), the user can enter aggregated weight and fuel values. These values are then used by the system to calculate figures such as zero-fuel weight, takeoff weight and estimated landing weight. Both the user-defined and calculated values can also be used by other G3000 sub-systems that require weight and/or fuel data.

The weight and fuel feature is enabled by default and is a required part of the G3000 avionics. Some airplane installations may opt to enable the optional weight and balance feature, which extends the weight and fuel feature with additional capabilities, such as the ability to define and load individual weight stations and carry out center-of-gravity (CG) calculations. To learn more about the weight and balance feature, please refer to this page. The next section describes how to configure the base weight and fuel feature.

Configuring Weight and Fuel​

Weight and fuel configuration is done through panel.xml. Configuration uses the <Weights> tag.

There are six values that all must be defined using child tags under <Weights>:

TagDescription
<BasicEmpty>The airplane's basic empty weight. This is the airplane's weight with no crew, no passengers, no cargo, and zero usable fuel.
<MaxZeroFuel>The airplane's maximum allowed zero-fuel weight. The airplane's zero-fuel weight is the airplane's total (gross) weight minus usable fuel.
<MaxRamp>The airplane's maximum allowed ramp weight. The airplane's ramp weight is the airplane's total (gross) weight "at the ramp" - when it is loaded with crew, passengers, cargo, and initial fuel and before it has burned any fuel.
<MaxTakeoff>The airplane's maximum allowed takeoff weight. The airplane's takeoff weight is the airplane's total (gross) weight at takeoff.
<MaxLanding>The airplane's maximum allowed landing weight. The airplane's landing weight is the airplane's total (gross) weight at landing.
<MaxPax>The maximum number of passengers allowed in the airplane. Crew (including pilots and flight attendants) are not counted as passengers.

The basic empty weight value is used to initialize the "Basic Empty Weight" field in the GTC Weight and Fuel page when the airplane is first loaded. After this, the user is free to edit the field to any value they wish. If persistent user settings are enabled for the airplane, then any changes made to the field by the user are carried over to future flight sessions. If persistent user settings are disabled, then the field is reset to the value defined in panel.xml at the start of each flight session.

The four maximum weight values (zero-fuel, ramp, takeoff, and landing) are used by the system to trigger alert states when the associated calculated weight value exceeds the maximum. The maximum passenger count value is used to constrain the range of values the user is allowed to enter for the GTC Weight and Fuel page's "Passengers" field.

Example​

The following example shows how to configure weight and fuel in panel.xml:

<Performance>
  <Weights>
    <BasicEmpty unit='pounds'>3000</BasicEmpty>
    <MaxRamp unit='pounds'>5300</MaxRamp>
    <MaxTakeoff unit='pounds'>5200</MaxTakeoff>
    <MaxLanding unit='pounds'>4750</MaxLanding>
    <MaxZeroFuel unit='pounds'>4200</MaxZeroFuel>
    <MaxPax>4</MaxPax>
  </Weights>
</Performance>
