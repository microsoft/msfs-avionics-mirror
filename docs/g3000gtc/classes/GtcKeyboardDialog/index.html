Attempts to resolve the current request.

If this dialog searches for facilities, then the currently selected facility will be returned if one exists. If there is no selected facility, duplicate matches will attempted to be resolved if they exist. If neither a selected facility or duplicate matches exist, the request will be cancelled.

If this dialog does not search for facilities, the current input text is returned.
